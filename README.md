# Restaurant Menus Directory App #
This web based application displays the catalog of restaurants and their corresponding menu items with detailed desription of each. The app is built with the help of Python's Flask MVC Framework together with SQLAlchemy ORM. The website allows users to login using OAuth 2.0 Facebook and Google authentication providers. Also, users can perform CRUD operations on their own restaurants and menu items, and they can get API enpoints for JSON represantation of menus and restaurants.
